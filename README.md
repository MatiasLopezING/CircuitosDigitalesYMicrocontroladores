# Circuitos Digitales y Microcontroladores 📚🔋

Este repositorio contiene las entregas correspondientes a los Trabajos Prácticos (del **TP1 al TP4**) de la cursada de **Circuitos Digitales y Microcontroladores**. Estas prácticas fueron realizadas y entregadas de forma grupal (equipos de a dos). 

El enfoque principal de los trabajos es el desarrollo sobre la arquitectura AVR y simulación de circuitos.

## 🗂️ Estructura de Entregas

El repositorio está configurado de manera estricta para subir únicamente las carpetas de `Entregas/` dentro de cada Trabajo Práctico:
- **`Practica/tp1/Entregas/`**
- **`Practica/tp2/Entregas/`**
- **`Practica/tp3/Entregas/`**
- **`Practica/tp4/Entregas/`**

Cada entrega incluye:
- El código fuente en `C` para los microcontroladores (proyectos de Microchip / Atmel Studio).
- Las simulaciones de los circuitos correspondientes desarrolladas en Proteus (`.pdsprj`).

## 🛠️ Herramientas Utilizadas

- **Microchip Studio (Anteriormente Atmel Studio):** Entorno de desarrollo para compilación del código en lenguaje C.
- **Proteus (ISIS Professional):** Para el diseño esquemático y simulación integral del sistema.
- **Compilador C (AVR-GCC):** Toolchain nativo de compilación.

---
> **Nota de versión:** Se utiliza un archivo `.gitignore` estricto en formato de lista blanca que oculta el resto del contenido personal del repositorio (clases, borradores, bibliografía, etc.) para mantener enfocadas sólo las entregas grupales finales.
